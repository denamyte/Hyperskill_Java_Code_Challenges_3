Asynchronous email sending

Some operations like email sending are often performed in a background thread. It allows reducing the load for main threads.

There is the MailSender interface and the MockMailSender class, which simulate mail sending.

Implement asyncSend method by iterating through messages and sending them via sender. Use a single thread pool executor and do not forget to shutdown it.



Memory limit: 256 MB
Time limit: 8 seconds
